# mt_economic_common
A number of montrek apps for usage in economical contexts. Namely, the apps are:
- currency
      App for currecnies in the DB and fetching fx rates.
- country
      App for countries in the DB.
- credit_institution
      App for credit institutions in the DB.

# Installation guide
```bash
Clone this repository in the montrek folder.
Add these entries to the INSTALLED_APPS environment parameter in you .env file:

```
mt_economic_common.credit_institution mt_economic_common.currency mt_economic_common.country 
```
